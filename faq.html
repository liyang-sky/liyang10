常见问题

遇到问题？先在这里搜索一下，或许已经有答案。

连接与信号

信号

开机后无画面怎么办？

1. 检查电源指示灯是否亮起，确认电源连接正常。
2. 按遥控器 INPUT 键，确认选择了正确的输入源。
3. 检查信号线是否插紧，尝试更换信号线。
4. 确认信号源设备已正常输出信号。

SDI

SDI 信号不稳定/花屏？

1. 检查 SDI 线缆长度是否在规格范围内（12G-SDI 建议 ≤ 50m）。
2. 确认线缆为合格的 75Ω 同轴电缆。
3. 检查接口是否有松动或氧化，重新插拔。
4. 尝试降低信号格式（如从 4K 降为 HD）测试线缆质量。

输入

HDMI 和 SDI 可以同时显示吗？

可以。HX3110 支持多画面显示模式，可在 OSD 菜单中设置「多画面显示」→ 选择不同输入源分配到各个画面区域。

画面与色彩

色彩

色彩偏色如何校准？

1. 使用专业校准探头（如 Klein K-10A）连接监视器。
2. 进入 OSD → 系统 → 自动校准，按提示操作。
3. 校准完成后保存到用户预设中。
4. 建议每 1-3 个月校准一次，保持色准。

HDR

HDR 模式下画面过亮/过暗？

1. 确认 HDR 模式设置正确（PQ 或 HLG）。
2. 检查素材的 HDR 元数据是否正常传输。
3. 调节「峰值亮度限制」参数。
4. 如果是 SDR 素材误设为 HDR 模式，请切回 SDR 模式。

设置

如何确认当前显示的是正确的色彩空间？

按遥控器 INFO 键，屏幕右上角会显示当前色彩空间、伽马曲线、HDR 模式等信息。也可在 OSD → 状态 中查看详细参数。

设置与操作

系统

如何恢复出厂设置？

进入 OSD → 系统 → 初始化 → 恢复出厂设置。注意：此操作会清除所有用户预设和校准数据，建议先备份重要配置。

固件

固件如何升级？

1. 从官网下载最新固件到 U 盘。
2. 将 U 盘插入监视器 USB 接口。
3. 进入 OSD → 系统 → 固件升级 → 选择 U 盘中的固件文件。
4. 按提示完成升级，期间请勿断电。

遥控

遥控器失灵怎么办？

1. 检查遥控器电池电量，更换新电池。
2. 确认遥控器与监视器之间没有遮挡。
3. 尝试重新配对遥控器（OSD → 系统 → 遥控器设置）。
4. 可使用监视器面板上的物理按键临时操作。
