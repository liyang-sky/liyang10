功能详解

BVM-HX3110 各项功能的详细说明与操作路径

1通道与输入源设置

HX3110 支持为每个通道独立配置输入信号和画面参数，通过 CH SELECT 按钮快速切换。

输入接口信号类型通道数
SDI IN 1-412G/6G/3G/HD-SDISingle Link: 4ch / Dual Link: 2ch / Quad Link: 1ch
HDMI INHDMI (含 HDCP)1ch
IP Input 1/3SMPTE ST 2110 (4K/2K)2ch
IP Input 2/4SMPTE ST 2110 (2K)2ch

操作路径：User Preset Setting → Ch. Setting → Input Select

4K/2K SDI Input1 和 Input3 自动区分 4K 和 2K 信号。IP 输入需预先在 IP Input Config. 中配置。

2EOTF（伽马曲线）设置

根据输入信号的伽马特性选择对应的 EOTF，确保正确还原画面亮度层次。

EOTF 选项适用场景
2.2 / 2.4 / 2.6 / CRTSDR 内容制作
2.4(HDR)HDR 显示模式
S-Log3Sony S-Log3 素材监看
SMPTE ST 2084PQ HDR 标准（电影/高端制作）
ITU-R BT.2100(HLG)HLG HDR 标准（广播电视）
S-Log3(Live HDR)Sony Live HDR 工作流专用

操作路径：User Preset Setting → Ch. Setting → EOTF

选择 HLG 时，可为每个峰值亮度档位（400~4000 nit）单独设置 System Gamma。选择 S-Log3(Live HDR) 时，用于评价与传统 SDR 环境监看的兼容性。

3色彩空间与传输矩阵

选择与输入信号匹配的色彩空间，确保色域正确还原。

色彩空间典型用途
ITU-R BT.709高清/标清广播电视
S-Gamut / S-Gamut3Sony 摄影机广色域素材
S-Gamut3.CineSony 数字电影摄影机
DCI-P3数字影院放映
ITU-R BT.20204K/HDR 超高清广播
SMPTE-C / EBU传统标清制式兼容

传输矩阵：Color Space 为 BT.2020 时自动选 BT.2020 矩阵；为 BT.709 时自动选 BT.709 矩阵；其他情况需手动匹配信号源的矩阵设置。

操作路径：User Preset Setting → Ch. Setting → Color Space / Transfer Matrix

4色温与色彩调节

提供标准色温预设和手动微调功能。

色温预设说明
D65 / D93 / D60 / D50标准 daylight 色温（已针对 Sony 监视器做光谱偏置补偿）
DCI / DCI XYZ数字影院标准（XYZ 格式信号固定为此）
User1 ~ User10自定义色温，通过 R/G/B Gain 和 Bias 手动调整

快捷调节：按前面板 MANUAL 按钮后，转动 CHROMA（色度）、BRIGHTNESS（亮度）、CONTRAST（对比度）旋钮实时调节。默认值 1000/0/400。

操作路径：User Preset Setting → Ch. Setting → Color Temp / Chr./Bright./Cont.

5波形图与监测工具

内置专业波形监视器，无需外接设备即可分析信号。

工具功能
WFM（波形图）亮度电平分布监测，支持 Line Select 定位单线
Vector Scope（矢量图）色彩相位与饱和度分析
CGS（色域图）色域范围可视化
False Color（伪色）亮度区间映射为不同颜色，快速曝光判断
Gamut Marker（色域标记）超出色域范围的部分以斑马纹显示
Camera Focus（聚焦辅助）高亮显示锐利边缘，辅助对焦

操作路径：Monitoring Tool 菜单 → Scopes / WFM/VS/CGS / False Color 等

注：WFM/VS/CGS 仅支持分辨率 ≥1280×720 的信号；Camera Focus 仅支持 ≥1920×1080。

6多画面显示

单屏同时监看 2 路或 4 路信号，对比不同环节画面。

模式显示方式适用场景
Quad View（四画面）2×2 网格，显示 A/B/C/D 四路输入多机位监看（不支持 4K 信号）
Side by Side（双画面）左右并排，显示 A/B 两路输入调色前后对比、不同版本对比

操作路径：Multi View 菜单 → Quad View / Side by Side

Quad View 和 Side by Side 模式下，画面 A 的音频信号被输出，时间码也仅显示画面 A 的。

7标记器与辅助显示

提供多种画面辅助标记，帮助构图和安全区确认。

功能说明
Aspect Marker画面比例标记（可自定义宽高比）
Area Marker 1/2自定义区域标记框
Center Marker中心十字标记
Asp. Blank.-Half / -Black非标准比例画面的遮幅显示
Grid Display网格线叠加，辅助构图对齐
Time Code时间码显示（支持 VITC/LTC）
Closed Caption隐藏式字幕显示（仅 HDMI/SDI Single Link）

操作路径：User Preset Setting → Marker Preset / Monitoring Tool → Grid Display / Metadata/Textdata → Time Code
