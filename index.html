01 · 核心技术

TRIMASTER HX™ 架构

索尼专有的设计架构,充分发挥专业平板显示器的全部性能。核心科技带来业界领先的色彩精度、精准成像与画质一致性。

双层防反射 LCD 面板

BVM-HX3110 配备索尼设计的双层防反射 LCD 面板,结合索尼专有信号处理技术,支持更高的峰值亮度、更快的像素响应与更宽的视角。EOTF 从深黑到高光的全范围准确跟踪,使其成为 BVM-HX310 之后专业用户信赖的参考标准。

  • ✓
    双层 LCD 结构显著降低环境光反射,提升对比度
  • ✓
    SORPLAS™ 再生塑料兼顾可持续发展与高性能
  • ✓
    防指纹涂层轻松擦除运输/安装时意外留下的指纹
02 · 核心特性

为专业制作而生的每一处突破

从 4000 cd/m² 峰值亮度到集成 IP 支持,8 大核心特性重新定义 4K HDR 主控监视器。

01

4000 cd/m² 峰值亮度

4K HDR 主控监视器的色彩还原度达到新高度。调色师将看到微妙细节、柔和亮光与丰富色彩,即使在以往细节缺失的图像区域也能呈现深邃黑色与流畅的亮光衰减。

D65 典型值 · 10% 白窗
02

快速像素响应

通过 BVML-F10 许可证激活的快速像素响应功能,非常适合体育直播和音乐制作。平移与倾斜运动显示更少模糊,滚动滚动条更易于阅读和检查。

需 BVML-F10 许可证
03

45° 宽视角

借助索尼光学显示屏材料,新型主控监视器的实用观看角度拓宽到 45°,不影响丰富的对比度或高色彩还原度。多名工作人员可同时进行审片工作。

0-45° 性能持续优化
04

改进的防反光

先进的防反光处理采用光波抵消技术,与音频降噪技术原理相似。消除投射到屏幕上的环境光,带来更出色的对比度、更清晰的图像,对分辨率影响极小。

双层 AR 涂层
05

集成 IP 支持

无缝集成到 IP 制作,支持 ST2110-20/21/22/30/40、NMOS IS04/IS05、净切换、无中断故障切换和 PTP。Web 菜单让 IP 接口配置变得简单,支持直接 IP 信号输入,无需 IP-SDI 转换器。

ST2110 · NMOS · PTP
06

四分屏 + Pixel Zoom

即时比较多达四组自定义显示设置,包括 EOTF、色彩空间、传输矩阵、色温、对比度、亮度和色度。V2.0 固件引入像素缩放,最大 300% 无损放大,用于死像素和焦点精确检查。

300% 无损放大
07

用户 3D LUT

通过按钮即可检查不同的 LUT。从 U 盘或其他 EOTF 加载自定义 LUT 进行调色前比较。高质量四面体插值让您能使用 LUT 进行 HDR-SDR 转换,与原片并排查看。

四面体插值算法
08

HDR-SDR 转换器

实现 4K/HD HDR 视频和 HD SDR 视频的同步制作。可选购许可证,当监看 SDR 或 HDR 信号时,承担 SDR 调色工作,使图像不存在饱和的亮光区域或挤压的暗光区域。

需 BVML-H10/S10/T10

04 · 可选许可证

按需扩展制作能力

6 款 BVML 系列许可证,从快速响应到 IP 解码,按工作流灵活配置。

F10

BVML-F10

快速响应许可证

激活 BVM-HX3110 的快速像素响应功能,大幅减少运动模糊。非常适合体育直播和音乐制作,滚动滚动条更清晰可读。

  • 体育赛事直播
  • 音乐节目制作
  • 动态画面监看
H10

BVML-H10

HDR-SDR 转换输出许可证

同时支持索尼 HDR-SDR、3D LUT 和 4K-HD 转换。在监看 SDR 或 HDR 信号时,自动承担 SDR 调色工作,确保图像亮光区域不过曝、暗部细节不丢失。

  • HDR/SDR 同步制作
  • 3D LUT 烘焙输出
  • 4K 到 HD 下变换
S10

BVML-S10

信号转换输出许可证

支持 3D LUT 和 4K-HD 转换输出。屏幕显示和转换输出可独立工作,确保切换显示时仍依赖同一信号输出。可对 IP 输入信号应用转换。

  • 3D LUT 处理
  • 4K-HD 转换
  • IP 信号处理
T10

BVML-T10

3D LUT 输出许可证

仅支持 3D LUT 转换。适合只需 LUT 烘焙输出、不需要 HDR-SDR 全套转换的场景。提供高性价比的 LUT 工作流。

  • 纯 LUT 烘焙
  • 现场调色预览
  • 成本敏感项目
JD10

BVML-JD10

JPEG XS 解码器许可证

为 IP 工作流添加 JPEG XS 解码能力。支持低延迟、轻量化的 IP 视频传输,适合远程制作和分布式工作流。

  • IP 远程制作
  • 低延迟传输
  • 分布式工作流
SN10

BVML-SN10

SNMP 许可证

为监视器添加 SNMP 网络管理协议支持,便于集成到大型设备监控系统中,实现远程状态查询、告警和参数配置。

  • 系统集成
  • 远程监控
  • 设备统一管理
