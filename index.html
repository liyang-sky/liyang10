TRIMASTER HX™ 架构

索尼专有的设计架构,充分发挥专业平板显示器的全部性能。核心科技带来业界领先的色彩精度、精准成像与画质一致性。

双层防反射 LCD 面板

02 · 核心特性

为专业制作而生的每一处突破

从 4000 cd/m² 峰值亮度到集成 IP 支持,8 大核心特性重新定义 4K HDR 主控监视器。

01

4000 cd/m² 峰值亮度

4K HDR 主控监视器的色彩还原度达到新高度。调色师将看到微妙细节、柔和亮光与丰富色彩,即使在以往细节缺失的图像区域也能呈现深邃黑色与流畅的亮光衰减。

D65 典型值 · 10% 白窗
02

快速像素响应

通过 BVML-F10 许可证激活的快速像素响应功能,非常适合体育直播和音乐制作。平移与倾斜运动显示更少模糊,滚动滚动条更易于阅读和检查。

需 BVML-F10 许可证
03

45° 宽视角

借助索尼光学显示屏材料,新型主控监视器的实用观看角度拓宽到 45°,不影响丰富的对比度或高色彩还原度。多名工作人员可同时进行审片工作。

0-45° 性能持续优化
04

改进的防反光

先进的防反光处理采用光波抵消技术,与音频降噪技术原理相似。消除投射到屏幕上的环境光,带来更出色的对比度、更清晰的图像,对分辨率影响极小。

双层 AR 涂层
05

集成 IP 支持

无缝集成到 IP 制作,支持 ST2110-20/21/22/30/40、NMOS IS04/IS05、净切换、无中断故障切换和 PTP。Web 菜单让 IP 接口配置变得简单,支持直接 IP 信号输入,无需 IP-SDI 转换器。

ST2110 · NMOS · PTP
06

四分屏 + Pixel Zoom

即时比较多达四组自定义显示设置,包括 EOTF、色彩空间、传输矩阵、色温、对比度、亮度和色度。V2.0 固件引入像素缩放,最大 300% 无损放大,用于死像素和焦点精确检查。

300% 无损放大
07

用户 3D LUT

通过按钮即可检查不同的 LUT。从 U 盘或其他 EOTF 加载自定义 LUT 进行调色前比较。高质量四面体插值让您能使用 LUT 进行 HDR-SDR 转换,与原片并排查看。

四面体插值算法
08

HDR-SDR 转换器

实现 4K/HD HDR 视频和 HD SDR 视频的同步制作。可选购许可证,当监看 SDR 或 HDR 信号时,承担 SDR 调色工作,使图像不存在饱和的亮光区域或挤压的暗光区域。

需 BVML-H10/S10/T10

Resolve

软件与输出设置

  • 安装并启用 Blackmagic Desktop Video / UltraStudio 驱动。

  • 确认接入 12G/6G/3G/HD-SDI 输入端口,并选择对应通道。

  • 预热约 30 分钟后再做严肃色彩判断。
AI + FFmpeg

片源格式不对时怎么处理

工程师不需要手写 FFmpeg 参数。先读取片源信息,再把网页里的标准提示词复制给 AI,由 AI 生成可在 Windows CMD 直接执行的转码命令。

01

使用 MediaInfo 或实验室现有工具,导出编码、封装、帧率、分辨率、色彩、音频等完整信息。

02

复制文件路径

准备视频文件完整路径。路径里有空格或中文时,后续命令必须使用英文双引号包住。

03

复制 AI 提示词

把 DaVinci 项目设置、文件路径和视频信息粘贴到提示词对应位置。

04

执行命令

将 AI 输出的命令粘贴到 CMD,或保存为 .bat 文件后执行。正式对比前先用短片段测试。

你只需要准备什么

  • DaVinci Resolve 项目设置:分辨率、帧率、SDR/HDR、色彩管理或色彩空间。
  • 片源文件完整路径,例如 D:\project\shot01.mov。
  • 视频信息软件导出的完整文本,不要只复制一两行摘要。

拿到命令后怎么做

  • 确认输入路径和输出路径正确,不要覆盖原片源。
  • 如果 AI 判断只需换封装,优先使用无损封装转换命令。
  • 质量对比建议输出 ProRes 422 HQ 或 DNxHR HQX 10-bit 这类中间格式。

教程速查

许可证使用教程

工程师第一次使用时,先在 Administrator > License 中确认 Activated License,再按对应教程检查输入、功能和输出。

BVML-F10 · 快速响应

  1. 确认 Activated License 中有 BVML-F10。
  2. 选择当前 SDI 输入通道,播放快速平移、滚动字幕或高速运动片段。
  3. 在监视器菜单中启用 Fast Response / 快速视频响应相关项目。
  4. 比较开启前后的拖影,并记录本项目是否启用该状态。

BVML-H10 · HDR-SDR 转换输出

  1. 确认 Activated License 中有 BVML-H10。
  2. 接入 DaVinci / UltraStudio 的 SDI 输出,设置输入 EOTF、Color Space 和 Transfer Matrix。
  3. 进入 HDR-SDR Conversion,选择或检查转换预设。
  4. 如需下游输出,进入 Enhanced Monitor Out 检查 SDI 输出格式。
  5. 注意 Enhanced Monitor Out 不能输出 HDMI 输入信号。

BVML-S10 · 信号转换输出

  1. 确认 Activated License 中有 BVML-S10。
  2. 确认当前输入为 SDI 或 IP 流,不要用 HDMI 输入测试 EMO 输出。
  3. 进入 Enhanced Monitor Out,选择需要的输出格式。
  4. 如需 LUT 效果,先在通道设置中启用 User LUT,再从 EMO 接出 SDI 验证。

BVML-T10 · 3D LUT 输出

  1. 确认 Activated License 中有 BVML-T10。
  2. 通过 USB 记忆棒导入项目确认过的 User LUT。
  3. 在目标通道中启用 User LUT 并选择正确 LUT 槽位。
  4. 从 Enhanced Monitor Out 接出 SDI,检查 LUT 输出效果。

BVML-JD10 · JPEG XS IP 解码

  1. 确认 Activated License 中有 BVML-JD10。
  2. 连接 LAN1/LAN2 的 25G IP 网络,并确认 SFP28、PTP、ST2110 参数已配置。
  3. 在 Web 菜单或本机菜单中检查 ST2110 Video/Audio、PTP、Multicast 或 NMOS。
  4. 选择 JPEG XS IP 流作为输入源,确认画面、音频和同步状态。

BVML-SN10 · SNMP 远程监控

  1. 确认 Activated License 中有 BVML-SN10。
  2. 确认监视器已接入实验室管理网络,IP 地址和权限由实验室统一维护。
  3. 打开 Web 菜单并进入 SNMP 设置,配置管理端和访问参数。
  4. 在监控系统中添加 BVM-HX3110,触发状态查询并确认能读取设备状态。
