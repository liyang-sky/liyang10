快速上手

4 个步骤完成从接线到正常监看。HX3110 具备自动识别功能，大部分参数不需要手动设置。

步骤 1：连接电源与信号线

电源连接

将电源线插入监视器背面板 AC IN 插孔，然后将交流插头固定器安装到电源线上。

SDI 信号连接（最常用）

将 SDI 线缆接入背面板 SDI IN 连接器（BNC）。
支持 Single Link（4 通道）、Dual Link（2 通道）、Quad Link（1 通道），自动识别。

HDMI 信号连接

将 HDMI 线缆接入背面板 HDMI IN 连接器。4K 信号请使用 3 米以内带 Premium High Speed 标识的线缆。

IP 信号连接（HX3110/HX1710N）

将 SFP28 收发器模块装入 LAN 1/LAN 2 插槽，接入 IP 流信号。需预先在菜单中配置 IP 输入参数。

[背面板接口示意图]
提示：12G-SDI/6G-SDI 建议使用 Canare L-5.5CUHD 或同等电缆，长度不超过 70 米。

步骤 2：开机

打开主电源

按下背面板主电源开关（），前面板电源指示灯亮为红色。

打开监视器

按下前面板 （电源）开关，指示灯由红变绿，屏幕点亮。

注意：无输入信号持续 60 分钟会自动关机，可在 System → System Setting → Auto Power Down 中修改。

步骤 3：选择通道与输入源

切换通道

按前面板 CH SELECT 按钮，旋转 SELECT/ENTER 控制钮选择通道，按控制钮确认切换。

配置输入源

进入 User Preset Setting → Ch. Setting → Input Select，选择对应的输入接口：

  • 4K/2K SDI Input1 / Input3 — 自动识别 4K/2K
  • 4K SDI Input1,2,3&4 — Quad Link 4K
  • 2K SDI Input1~Input4 — 单通道 2K
  • HDMI — HDMI 信号
  • 4K/2K IP Input1 / Input3 — IP 流（需预先配置）

步骤 4：打开自动识别（最关键）

HX3110 内置自动识别功能，打开后监视器会根据输入信号自动配置大部分画面参数，不需要手动逐项设置。

① VPID/HDMI Auto — 自动识别信号格式

打开后，监视器自动从输入信号中读取并设置以下参数：

  • EOTF（伽马曲线）— 自动识别 PQ / HLG / S-Log3 / 2.4 等
  • Color Space（色彩空间）— 自动识别 BT.709 / BT.2020 / DCI-P3 等
  • Transfer Matrix（传输矩阵）— 自动匹配
  • RGB/YCC Range — 自动识别 Full / Limited

设置路径：User Preset Setting → Ch. Setting → VPID/HDMI Auto → On

② SR Live Auto — HDR 制作自动配置

如果你在做 HDR 内容（尤其是 Sony 摄影机拍摄的素材），打开此选项后：

  • 自动读取信号中的 SR Live Metadata
  • 自动配置 HDR→SDR 转换所需的 EOTF、Color Space、Transfer Matrix
  • 自动应用转换预设（Conv. Preset）

设置路径：User Preset Setting → Ch. Setting → SR Live Auto → On

注：当 VPID/HDMI Auto 和 SR Live Auto 同时为 On 时，SR Live Auto 优先。

③ SDI 输入自动识别

进入 System → System Setting → SDI In. Setting，将以下两项设为 Auto：

  • Image Division — 自动识别 4K 信号的图像分割方式（2SI / Square）
  • Signal Format — 自动识别信号采样格式（422 YCbCr 10bit / 444 RGB 10bit 等）

设置路径：System → System Setting → SDI In. Setting → Image Division: Auto / Signal Format: Auto

一句话总结：打开 VPID/HDMI Auto 和 SR Live Auto，接好线、选对输入源，监视器自己就能认出来信号是什么格式，画面参数自动配好。

步骤 5：确认画面正常

打开自动识别后，画面应该已经正常显示。检查以下几点：

  • 画面是否有信号？没有信号检查线缆连接和输入源选择
  • 色彩是否看起来正常？如果偏色严重，可能是自动识别失败，需要手动检查 EOTF 和 Color Space
  • 亮度是否合适？环境光较强时可适当提高 Brightness，但不要影响调色准确性

如需查看当前自动识别的参数值，按 MENU 键进入 User Preset Setting → Ch. Setting，各项参数会显示当前设置。

什么时候需要手动设置？
1. 信号没有嵌入 VPID/HDMI 信息帧（部分老旧设备）
2. 信号信息不明确或标记错误
3. 使用特殊 LUT 或自定义色彩空间
4. 需要与特定标准严格匹配（如调色监看需固定 DCI-P3 / Gamma 2.6）
5. 做 HDR→SDR 转换时，Conv. Preset 需要手动微调
手动设置方法详见 功能详解 页面。
快捷操作：将常用功能分配给前面板 F1-F12 功能按钮，一键切换通道、开关波形图、切换标记等。设置路径：F Key Setting → Monitor (BVM-HX3110)。
