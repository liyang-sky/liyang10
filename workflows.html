场景工作流

针对不同应用场景，提供推荐的参数配置方案。可直接在监视器上按表格设置。

调色监看

影院后期调色、广告调色。目标：色彩准确还原，与影院放映端一致。

EOTF2.6
Color SpaceDCI-P3
Transfer MatrixITU-R BT.709（DCI-P3 时）
Color TempDCI 或 DCI XYZ（XYZ 格式信号）
Peak Lumi. Config.根据调色室标准选择（通常 100 nit 对应约 400 档）
Chr./Bright./Cont.1000 / 0 / 400（默认）
菜单路径：User Preset Setting → Ch. Setting → EOTF / Color Space / Color Temp

HDR 内容制作（PQ）

HDR 电影、高端广告制作。发挥 4000 nit 峰值亮度，精确还原 PQ 曲线。

EOTFSMPTE ST 2084
Color SpaceITU-R BT.2020
Transfer MatrixITU-R BT.2020
Color TempD65
Peak Lumi. Config.1000 nit（母带 1000 nit）或 4000 nit（母带 4000 nit）
Brightness 调节支持 ITU-R BT.814-4 标准亮度调节
提示：长时间观看高亮画面易视觉疲劳，建议定时休息。HDR 显示时屏幕表面会发热，勿触摸。

HDR 内容制作（HLG）

广播电视 HDR 制作。HLG 兼容 SDR 显示设备，适合直播和电视播出。

EOTFITU-R BT.2100(HLG)
Color SpaceITU-R BT.2020
Transfer MatrixITU-R BT.2020
Color TempD65
Peak Lumi. Config.根据播出端亮度选择（如 1000 nit）
System Gamma根据 Peak Lumi. 自动关联（如 1000 nit 默认 1.200）
菜单路径：User Preset Setting → Ch. Setting → EOTF → 选择 HLG 后设置 System Gamma

现场制播 / SDR 播出

直播、晚会、体育赛事等传统广播电视制作。稳定、低延迟、兼容现有 SDR 链路。

EOTF2.4
Color SpaceITU-R BT.709
Transfer MatrixITU-R BT.709
Color TempD65
Peak Lumi. Config.400 或 500（对应约 100 nit）
输入信号SDI（Single Link 3G/HD-SDI）
建议：现场光线较强时，可适当提高 Chr./Bright./Cont. 中的 Brightness 值，但调色监看时应保持标准亮度。

Sony Live HDR 工作流

Sony 倡导的 S-Log3 Live HDR 制作。监视器显示添加系统伽马的 S-Log3 信号，兼顾 HDR 监看与 SDR 兼容性评价。

EOTFS-Log3(Live HDR)
Color SpaceS-Gamut3 或 S-Gamut3.Cine（匹配摄影机设置）
Color TempD65
Peak Lumi. Config.根据现场需求选择（通常 1000~4000 nit）
VPID/HDMI AutoOn（自动识别 SR Live Metadata）
SR Live AutoOn（自动应用 HDR→SDR 转换设置）
说明：S-Log3(Live HDR) 用于调节 HDR 摄像机图像时执行恰当且舒适的图像表现，同时评价与传统 SDR 环境监看的兼容性。
